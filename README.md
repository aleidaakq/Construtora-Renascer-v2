# 🏗️ Construtora Renascer

Website institucional desenvolvido para a **Construtora Renascer**, com foco em apresentar a empresa, seus serviços, portfólio e facilitar o contato com novos clientes.

O projeto foi desenvolvido utilizando apenas tecnologias web nativas, priorizando performance, organização do código e uma arquitetura CSS escalável.
